# 🫀 Heart Disease Machine Learning & Data Pipeline Project

A complete end-to-end Data Science and Machine Learning project demonstrating Data Cleaning, Feature Engineering, Regression, Classification, and Comprehensive Model Comparison using a Synthetic Heart Disease Dataset.

---

## 📌 Project Overview

This project builds a full-stack Machine Learning pipeline designed to handle real-world medical data challenges. It spans across:
1. **Data Preprocessing & Cleaning:** Handling missing values, bad clinical data, and feature encoding.
2. **Regression Analysis:** Continuous prediction of patient age using clinical attributes.
3. **Classification Modeling:** Categorical prediction of heart disease risks.
4. **Model Performance Evaluation:** Comparative analysis across models and performance metrics.
